Какую проблему решает приложение?
Приложение позволяет получить инофрмацию о животных, 
соджержащихся в приюте

Кто пользователи?
Человек, которых хочет забрать себе питомца из приюта

Какие основные функции (MVP)?
1. Предоставление информации в виде списка о животном
2. Функция забрать животное из приюта предоставляет контактные данные приюта
